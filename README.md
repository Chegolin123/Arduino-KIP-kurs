**Образовательная платформа колледжа КИП ФИН** для изучения Arduino.  
Структурированные курсы, каталог компонентов, тестирование и отслеживание прогресса студентов.

---

## 🚀 Функциональность

### Для студентов:
- 📚 **Библиотека курсов** — выбор курса по уровню сложности
- 📖 **Учебные материалы** — главы и разделы с видео, изображениями и текстом
- 📝 **Тестирование** — тесты в конце каждой главы
- 📊 **Профиль с метриками** — прогресс, результаты тестов, успеваемость
- 🔧 **Каталог компонентов** — подробные карточки элементов Arduino

### Для администраторов:
- 🛠 **Управление курсами** — создание и редактирование курсов
- 📑 **Управление главами** — добавление разделов с визуальным редактором
- ✅ **Управление тестами** — создание вопросов с вариантами ответов
- 👥 **Управление студентами** — фильтрация по группам, просмотр статистики
- 📦 **Управление каталогом** — CRUD товаров
- 📥 **Экспорт/импорт** — Excel, Word, JSON
